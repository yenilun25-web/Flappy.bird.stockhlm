
# 🐦 Flappy Bird Game

Tap to fly! Avoid the pipes and score as high as you can.

## 🎮 How to Play
- Tap anywhere on the screen to make the bird jump.
- Pass through the gaps between pipes.
- Each pipe you pass = 1 point.
- If you hit a pipe or the ground, game over.
